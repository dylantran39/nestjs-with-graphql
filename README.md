## NestJS with Graphql

Structure project with NestJS and Graphql

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

# generate schema
$ npm run generate:schema

# seed cats data
$ npx nestjs-command seed:cats
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

With the server running, visit http://localhost:3000/graphql
